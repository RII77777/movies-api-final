## Project Overview
A full-stack movie database application with user authentication, role-based access control (RBAC), and reviews system. Built with Node.js, Express, MongoDB, and vanilla JavaScript frontend.

## Features
- **User Authentication**: Register/Login with JWT tokens
- **Role-Based Access Control**: Admin users can manage movies
- **Movie Management**: CRUD operations for movies
- **Review System**: Users can post reviews for movies
- **Responsive Design**: Works on all device sizes
- **MongoDB Integration**: Persistent data storage with Mongoose

## Technology Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: JWT (JSON Web Tokens), bcrypt
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Middleware**: CORS, express-validator, custom auth middleware
- **Deployment**: Render (Backend), MongoDB Atlas (Database)
